# 🌟 Josiane Nikuze — Senior Systems Architect Portfolio

A premium, production-grade portfolio designed with a modern bento-grid aesthetic, fluid spring micro-animations, and technically excellent architecture. Built from the ground up for elite recruiters and enterprise collaborators to showcase a senior engineering persona.

---

## 🎨 Design Philosophy & Aesthetics

* **Minimalist High-Tech Theme**: Curated deep-space dark palette with active system glows, fully responsive and supported by a seamless, native light theme transition.
* **Bento Grid Layouts**: High-density component blocks inspired by modern SaaS interfaces, presenting skills, experience, and assets with structured efficiency.
* **Fluid Micro-Animations**: Smooth, physical spring-based animations powered by Framer Motion, elevating active states, hover transitions, and modal transitions.

---

## ⚡ Core Features

### 🛠️ Bento Grid Expertise Dashboard
* An icon-driven dashboard dividing skills into specialized domains (Architecture, Integration, Engineering).
* High-contrast monospace labels, interactive hover scaling, and clean structural highlights.

### 📄 Systems Analyst Resume Hub
* Interactive, custom modal preview system that avoids raw PDF browser forwards.
* Provides recruiters with direct actions to **Download**, **View**, or gracefully **Dismiss** the document without leaving the single-page application context.

### 💼 Case Study Modals
* Deep-dive documentation overlays for enterprise projects (e.g., *ExploreRwanda Ecosystem*, *SafeRide Platform*).
* Features embedded architecture logs, system workflows, and clickable technical asset indicators.

### ✉️ Premium Stacked Contact Hub
* A modern, dashboard-style 2-column stacked input grid designed to optimize layout balance.
* **Embedded Input Icons**: Beautiful Lucide symbols (`User`, `Mail`, `Tag`, `MessageSquare`) nested inside inputs, turning bright blue when focused.
* **Resilient Web3Forms integration**: Native asynchronous `fetch` submit handler with graceful simulated fallback systems for smooth recruiter demos.
* **Operational Success Screen**: Pulses a glowing emerald radar ring with custom checkmark feedback upon successful form submission.

### 🌐 Brand-Reactive Sitemap Footer
* Features a glowing **`JN`** initial brand mark with hover-aura highlights.
* **Hover-Tailored Social Cards**: Glassmorphic connect tiles that dynamically glow in their official platform brand colors (White for GitHub, Corporate Blue for LinkedIn, Sky Blue for Twitter/X).
* **Operational Status Badge**: A glowing system indicator reading `● ENGINEERED FOR EXCELLENCE` using green beacon keyframe pulse animations.

## 💻 Technology Stack

* **Core**: [React 18](https://react.dev/) (Hooks, State binding, dynamic theme providers)
* **Build Engine**: [Vite](https://vitejs.dev/) (Sub-millisecond Hot Module Replacement)
* **Animations**: [Framer Motion](https://www.framer.com/motion/) (Physics-based transition engines)
* **Styling**: [Vanilla CSS](https://developer.mozilla.org/en-US/docs/Web/CSS) (Tailored custom properties, keyframe cycles, layout calculations)
* **Icons**: [Lucide React](https://lucide.dev/) (Sharp vector UI markers)
* **API Delivery**: [Web3Forms](https://web3forms.com/) (Direct asynchronous email routing)

## 🚀 Getting Started

### 📋 Prerequisites
Ensure you have [Node.js](https://nodejs.org/) installed on your machine.

### ⚙️ Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/Josiane-Ni/My-Portifolio.git
   ```
2. Navigate to the project directory:
   ```bash
   cd My-Portifolio
   ```
3. Install dependencies:
   ```bash
   npm install
   ```

### 🛠️ Development Command
Start the Vite HMR server locally:
```bash
npm run dev
```
The server will boot up at `http://localhost:5173`.

### 📦 Production Bundle
Compile and optimize assets for static hosting:
```bash
npm run build
```
The production-ready assets will be generated in the `dist` directory.
